This is a quick tutorial for new users

Create Plan:

To create a new plan, click the create new plan tab in the navigation bar

View Plan:

There are three tabs for view plans options.

  • View available plans show all plans that you can see. Notice that the available plans are only those plans created by yourself, your friends on Facebook and Administrator. You can not view plans created by any strangers.

  • View my plans tab will show you the plans you have created.

  • View Join plans tab will show you the plans you have joined.

Filter

To search for a specific condition of travel plans, you can use the filter box.

  • The destination field will only show the plans that destination name contains the query string.

  • The depart after field will only show the plans that depart after the query time.

  • The return before field will only show the plans that return before the query time.

These three conditions can be combined to filter.

Join Plan

Click into the plan detail page, and click the join plan button.

Edit Plan

Click into the plan detail page, and click the edit plan button.

Delete Plan

Click into the plan detail page, and click the delete plan button.

Share Plan

Click into the plan detail page, and click the share plan button.
